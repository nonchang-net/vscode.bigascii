# bigASCII README

- `english`
	- convert ascii to minimap friendry big-ascii.
	- run with "BIG:todo" line, you get result.

- `japanese`
	- アスキー文字列を、minimapで表示可能なアスキーアートに変換します。
	- 「BIG:todo」と入力した行で実行すると、以下の結果を得られます。

```

result:

######    ####    ######      ####  
  ##    ##    ##  ##    ##  ##    ##
  ##    ##    ##  ##    ##  ##    ##
  ##    ##    ##  ##    ##  ##    ##
  ##      ####    ######      ####  

```
